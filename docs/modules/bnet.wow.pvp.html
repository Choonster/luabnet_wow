Module bnet.wow.pvp

Blizzard Battle.net Community Platform API Library.

Easily retrieve various types of data from Blizzard's API in the format of Lua tables. Implements the PVP Resources section of the API.

Functions

wow:GetArenaTeamInfo (realm, teamSize, team, locale, forceRefresh)
Retrieve arena team information.
See :SendRequest for return values. See :SendRequest and :SendRequestRaw for more information on the forceRefresh parameter.

Parameters:

  • realm (string) The realm of the team.
  • teamSize (number) The size of the team. For a 2v2 team, use 2 as the team size.
  • team (string) The name of the team.
  • locale (string, optional) The locale to retrieve the data in.
  • forceRefresh (boolean, optional) If true, send a request regardless of cached results.
wow:GetArenaLadder (battlegroup, teamSize, page, size, ascending, locale, forceRefresh)
Retrieve the Arena Ladder for the battlegroup.
See :SendRequest for return values. See :SendRequest and :SendRequestRaw for more information on the forceRefresh parameter.

Parameters:

  • battlegroup (string) The Battlegroup to query.
  • teamSize (number) The size of the team. For a 2v2 team, use 2 as the team size.
  • page (number, optional) Which page of results to return (defaults to 1)
  • size (number, optional) How many results to return per page (defaults to 50)
  • ascending (boolean, optional) Whether to return the results in ascending order. Defaults to true if omitted.
  • locale (string, optional) The locale to retrieve the data in.
  • forceRefresh (boolean, optional) If true, send a request regardless of cached results.
