wow:GetAuctionData (realm, forceRefresh)
Retrieve an auction data dump. Due to the extremely large size of these dumps, this request will often take a long time to process. Using a faster decompression library will speed this up slightly.
See :SendRequest for return values. See :SendRequest and :SendRequestRaw for more information on the forceRefresh parameter.

Parameters:

  • realm (string) The realm to get auction data for.
  • forceRefresh (boolean, optional) If true, send a request regardless of cached results.
