wow:GetBattlePetAbilityInfo (abilityID [, locale [, forceRefresh]])
Retrieve information about a Battle Pet Ability.

Parameters:

  • abilityID number The abilityID to query.
  • locale string The locale to retrieve the data in.
  • forceRefresh bool If true, send a request regardless of cached results.

Returns:

  1. bool success: Did the query succeed?
  2. tab result: The decoded JSON data.
  3. number code: The HTTP response status code. If no request was sent, this will be 304.
  4. string status: The full HTTP response status. If no request was sent, this will be "No request sent".
  5. table headers: The HTTP headers of the response. If no request was sent, this will be nil.
  6. number time: The number of seconds between the function being called and the results being returned, calculated with os.time().
  7. number clock: The number of seconds of CPU time used between the function being called and the results being returned, calculated with os.clock().
