Module bnet.wow.challengemode

Blizzard Battle.net Community Platform API Library.

Easily retrieve various types of data from Blizzard's API in the format of Lua tables. Implements the Challenge Mode section of the API.

wow:GetChallengeModeRealmLeaderboard (realm, locale, forceRefresh)
Retrieve the Challenge Mode Leaderboard for the realm.
See :SendRequest for return values. See :SendRequest and :SendRequestRaw for more information on the locale and forceRefresh parameters.

Parameters:

  • realm (string) The name or slug of the realm.
  • locale (string, optional) The locale to retrieve the data in.
  • forceRefresh (boolean, optional) If true, send a request regardless of cached results.
