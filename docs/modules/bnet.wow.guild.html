Module bnet.wow.guild

Blizzard Battle.net Community Platform API Library.

Easily retrieve various types of data from Blizzard's API in the format of Lua tables. Implements the Guild Resources section of the API.

wow:GetGuildProfile (realm, guild, fields, locale, forceRefresh)
Retrieve a guild profile table.
See :SendRequest for return values. See :SendRequest and :SendRequestRaw for more information on the locale and forceRefresh parameters.

Parameters:

  • realm (string) The realm of the guild.
  • guild (string) The name of the guild.
  • fields (string, optional) A list of comma-separated fields to query.
  • locale (string, optional) The locale to retrieve the data in.
  • forceRefresh (boolean, optional) If true, send a request regardless of cached results.
